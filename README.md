Bienvenue sur le dépôt de mon portfolio personnel. Ce projet n'est pas qu'une simple vitrine ; c'est une étude de cas sur la **performance**, l'**éco-conception** et l'**expérience utilisateur fluide**.

## Caractéristiques

* **React 19 & Vite** : Utilisation des dernières fonctionnalités pour une réactivité maximale.
* **Animations High-End** : Transitions fluides et effets de cascade avec **Framer Motion**.
* **Scores Lighthouse** : Intégration de jauges SVG animées pour illustrer les performances réelles des projets.
* **Éco-conception** : Optimisation des assets, lazy loading et architecture sobre.
* **Full Responsive** : Expérience optimisée du mobile au desktop 4K.

## Stack Technique

* **Frontend** : React 19, React Router 7.
* **Style** : CSS Modules (pour une isolation parfaite des styles).
* **Animations** : Framer Motion (Gestures, AnimatePresence, InView).
* **Icônes** : Lucide-React.
* **SEO** : React Helmet Async.

## Installation & Lancement

Si vous souhaitez explorer le code ou le faire tourner en local :

1.  **Cloner le projet**
    ```bash
    git clone https://github.com/ClemLy/Portfolio-2026.git
    cd portfolio-clementin
    ```

2.  **Installer les dépendances**
    > Note : Le projet utilise React 19. Si vous rencontrez des conflits de dépendances avec des librairies tierces, utilisez le flag suivant :
    ```bash
    npm install --legacy-peer-deps
    ```

3.  **Lancer en mode développement**
    ```bash
    npm run dev
    ```
